*** Option B: Editor-Agnostic Pre-Commit Git Hook
A private executable script inside the local repository folder at =.git/hooks/pre-commit= to rebuild changed vector graphics automatically right before a commit occurs:
#+begin_src bash
#!/bin/bash
# Find all modified or added .puml files in the diagrams source folder
PUML_FILES=$(git diff --cached --name-only | grep 'docs/diagrams/src/.*\.puml$')

if [ -n "$PUML_FILES" ]; then
    echo "Processing updated PlantUML diagrams..."
    for file in $PUML_FILES; do
        filename=$(basename "$file" .puml)
        plantuml -tsvg "$file" -o "../generated/"
        git add "docs/diagrams/generated/${filename}.svg"
    done
fi
#+end_src
